== Description ==

3-D Secure Payment Gateway By GBPrimePay, Accept Credit Card, QR Code and Bill Payment on WooCommerce store with [GBPrimePay Payments](https://www.gbprimepay.com/)


## Requirements

To use the GBPrimePay Payments, you'll first need to [register for an GBPrimePay account](https://www.gbprimepay.com/).

== Installation ==

= Manual installation =

1. Download the plugin.
2. Upload the plugin files to the `/wp-content/plugins/gbprimepay-payment-gateways/` directory.
3. Activate the plugin through the "Plugins" screen in menu.
4. Use the GBPrimePay on Main Top Menu > GBPrimePay Settings screen to configure the plugin.
